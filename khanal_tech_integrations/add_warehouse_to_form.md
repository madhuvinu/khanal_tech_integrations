# Steps to Add Warehouse Field to Form Layout

Since the warehouse field exists in the database but isn't showing in the form editor, follow these steps:

1. **In the Doctype Editor (Form Tab)**:
   - You should see an "Add field" button at the bottom of the form card
   - Click this button

2. **Search for warehouse**:
   - A search/dropdown should appear
   - Type "warehouse" or scroll to find it
   - Select "warehouse" from the list

3. **Position the field**:
   - After adding, drag it to position it after "Variant" and before "Grams SKU"
   - The order should be: Category → Item Code → Variant → **Warehouse** → Grams SKU → SKU Code

4. **Save**:
   - Click "Save" at the top right
   - The warehouse field should now appear on the form
